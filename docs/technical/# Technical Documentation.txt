## Technology Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: CSS Modules, Responsive Design
- **State Management**: React Context API
- **Internationalization**: Custom i18n implementation
- **Build Tool**: Vite
- **Package Manager**: npm
- **Deployment**: Plesk with Node.js
