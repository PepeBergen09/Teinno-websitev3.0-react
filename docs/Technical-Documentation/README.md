## Quick Reference

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite 7
- **Styling**: Tailwind CSS v4
- **Routing**: React Router v7
- **Animations**: Framer Motion
- **Form Handling**: React Hook Form

## Prerequisites

- Node.js 18+ 
- npm or yarn
- Git
- Basic knowledge of React and TypeScript
